# Exercise: Agent or Assistant?

**Module:** 04 — Agent vs Assistant

## Learner Objective

Correctly classify work scenarios as best suited for assistant mode or agent mode, and explain why the distinction matters.

## What You'll Do

Read four work scenarios and classify each one. Then pick the scenario that would make you most cautious about using agent mode and explain why.

## The Task

For each scenario below, say: **Assistant** (you guide each step) or **Agent** (it can act autonomously), and give one sentence explaining why.

**Scenario A:** "I want to understand the key differences between two vendor contracts before a negotiation tomorrow."

**Scenario B:** "I want the AI to go through our shared folder, find all documents from 2022, and create a summary index."

**Scenario C:** "I want to brainstorm a list of questions to ask a job candidate before an interview."

**Scenario D:** "I want the AI to draft and send a follow-up email to each person who attended last week's training."

Then answer this final question:

**Which of these four would you be most cautious about running in agent mode, and what one safeguard would you put in place before you did?**

## Completion Criteria

- [ ] Each scenario is classified as assistant or agent
- [ ] Each classification includes a one-sentence rationale
- [ ] The learner has named a specific safeguard for their most cautious scenario (not just "I'd be careful")
- [ ] You can state in one sentence what makes Scenarios B and D different from A and C

## Common Confusion

**"Can't the AI just do all of these?"** Yes, technically — but "can" is different from "should." The question is which requires you to stay in the loop and which is safe to hand off.

**"Scenario D seems fine — it's just email."** Sending communications on your behalf to other people is irreversible. Ask: what happens if the AI sends to the wrong list, uses the wrong tone, or includes content you didn't intend?

## Facilitator Note

Scenario D is a trap — it sounds routine but it's high-stakes agent behavior (sending communications to real people). If the learner classifies D as low-risk assistant work, probe: "What does 'agent' mean to you in this context?" Their answer will show whether the module's core concept landed.
